# Subscription Churn Prediction : Revenue-Weighted Retention Prioritisation

Predicting customer churn and quantifying expected revenue at risk for each account,
using the IBM Telco Customer Churn dataset (7,043 customers, 20 features).

## Business Problem and Context

Subscription churn is one of the highest-impact problems in recurring-revenue products.
This project predicts which customers are likely to cancel and ranks them by expected
monthly revenue at risk, turning a binary classifier into a prioritisation tool for
retention teams.

## Dataset

IBM Telco Customer Churn : 7,043 customer records, 20 features covering demographics,
services, contract type, billing, and a binary churn label. Overall churn rate: 26.5%.

Source: https://www.kaggle.com/datasets/blastchar/telco-customer-churn

## Tools

Python, Pandas, NumPy, Matplotlib, Seaborn, Scikit-learn, XGBoost, SHAP
